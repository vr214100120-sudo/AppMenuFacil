“Menú Fácil” es una aplicación móvil desarrollada con el propósito de optimizar el proceso de pedidos dentro de restaurantes mediante una experiencia digital moderna, rápida e intuitiva. La aplicación fue creada en Android Studio utilizando Kotlin y Jetpack Compose, herramientas que permitieron construir una interfaz dinámica, organizada y adaptable a dispositivos móviles Android. Además, el diseño visual fue previamente elaborado en Figma, permitiendo planificar la estructura y experiencia de usuario antes del desarrollo.

La aplicación está orientada tanto a clientes como a empleados del restaurante. Los clientes pueden ingresar su nombre y número de mesa, visualizar el menú digital, buscar productos y agregarlos a un carrito de compras para posteriormente confirmar su pedido. Por otro lado, los empleados cuentan con una vista especializada donde pueden consultar los pedidos activos y mantener un mejor control de las órdenes registradas.

“Menú Fácil” fue diseñada bajo principios de simplicidad y facilidad de uso, utilizando colores cálidos y una navegación lineal que reduce errores y mejora la interacción del usuario. El proyecto busca modernizar la atención dentro de restaurantes, agilizar el servicio y ofrecer una alternativa tecnológica eficiente para la gestión de pedidos dentro del entorno gastronómico.
